Защита Территории

1. Название проекта: «Защита территории»
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/301fa89f-75dd-4562-978a-0359bd145d2c" />

2. Жанр: Third-Person Shooter
3. Платформа: ПК
4. Управление: WASD, R, Q, C, Space, Shift, мышь
5. Цель игры: Уничтожить 10 волн врагов, защитив минимум 2 объекта
6. В игре присутствует возможность настроить качество графики, громкость звука, громкость музыки, яркость
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/d3df78e4-28c6-4401-90d6-a9be791c3f52" />

Элементы геймплея

1. Управление персонажем:
   1) Игрок управляет персонажем с помощью клавиш WASD для передвижения по уровню
   2) Игрок управляет камерой с помощью мыши для обзора окружения
   3) Игрок может бегать, приседать, перезаряжаться, стрелять, прицеливаться.
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/dba9d1a4-0659-4316-8adf-9e1e521361ef" />

2. Система здоровья: У персонажа есть уровень стресса, если шкала стресса поднимется до максимума то игра будет проиграна, на территории уровня есть 3 объекта, если у 2 объектов закончится прочность игра будет проиграна.
3. Враги-боты: В игре есть 3 вида врагов:
   1) Желтые - быстрые
   2) Красные - сильные
   3) Синие - прочные
Боты, увидев игрока, стреляют в него, иначе бегут и стреляют по случайному объекту
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/e50f67bd-26dc-4120-86e8-d12e71eabe36" />

4. Логика бонусов: На уровнях есть 4 вида бонусов, которые создаются на месте уничтоженного врага:
   1) Патроны - нужны чтобы убить ботов
   2) Аптечка - снижает стресс
   3) Аура - снижает стресс игрока и здоровье врага внутри неё
   4) Турель - стреляет по близко подошедшим врагам
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/88e22caa-a66d-46d3-b2e6-7ec5313e6006" />

5. Прокачка игрока: После каждой волны игрок может прокачать любой из 4 навыков - скорость, стрессоустойчивость, атаку, прочность объектов
